# H. K. Chemicals Industries Website

Official website for H. K. Chemicals Industries - Leading Sodium Hypochlorite Manufacturer in India.

## About

H. K. Chemicals Industries is a leading manufacturer and exporter of Sodium Hypochlorite (NaOCl) with 25+ years of excellence. Based in Dahej GIDC, Gujarat, India, with a production capacity of 3000+ MT/month.

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Smooth Scrolling**: Lenis
- **Icons**: Lucide React

## Features

- Award-winning modern design
- Premium animations and interactions
- Smooth scrolling experience
- Fully responsive (mobile-first)
- SEO optimized
- Fast loading with optimized images
- Contact form with validation
- Google Maps integration

## Getting Started

### Development

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
yarn build
yarn start
```

## Deployment

This website is optimized for deployment on Vercel:

1. Push code to GitHub repository: `workkevanshinaik-ui/hkchemicals-website`
2. Connect repository to Vercel
3. Deploy automatically
